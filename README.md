Algorithms
==========
A small package with various data structures and algorithms. Please open a bug
report if anything is working improperly! Currently, the following are
implemented:

Data Structures
---------------
Current data structures include:
+   Queue
+   Stack
+   Dequeue
+   Circular Buffer
+   LinkedList
+   Dynamic Array
+   Binary Heap
+   Binary Search Tree
+   Trie
+   Graph

Algorithms
----------
Current algorithms includes:
+   Quicksort with three variants
    +   Naive
    +   Randomized
    +   Median-of-medians
+   Merge sort
+   Insertion sort
+   Bubble sort
+   Bogosort
+   Bogobogosort
+   Selection sort
